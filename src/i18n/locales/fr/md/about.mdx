# À propos

<br />

Meme Studio est un site web simple et rapide pour créer et partager des mèmes. Une [liste de mèmes](/gallery) est fournie par défaut, mais vous pouvez également importer votre propre image.

<br />

Vous avez la possibilité de télécharger votre mème personnalisé sur votre appareil ou de le copier dans votre presse-papier. Nous ne sauvegardons pas votre mème dans notre base de données, veillez donc à ne pas oublier à exporter votre travail.

<br />

Si vous avez des suggestions, des commentaires ou si vous souhaitez signaler un problème, veuillez utiliser le [formulaire de feedback](https://forms.gle/KT2wHbobQwKLXDo1A). Si vous souhaitez qu'une image soit supprimée, veuillez me contacter : victor.delafouchardiere@gmail.com
